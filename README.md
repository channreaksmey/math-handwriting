# Math Handwriting Data Collection System

## Project Goal

Build a web application to collect handwriting samples from children solving math problems, storing rich stroke data (not just final answers) for future AI model training to evaluate problem-solving process rather than just correctness.

## Quick Start

```bash
cd python-backend
python run.py
# then
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
